# 关于`Echo`

**这是一款仅供安全测试学习的图形化一键接受反弹**shell**以及一些场景下的回连测试工具**

**本着白嫖至上的理念，请低调使用！**

**目前想法：**

1. [✔️] 将本地服务端口穿透至公网
2. [✔️] 用`go`实现`nc`接收反弹 shell，回连本地内网
3. [❌] 实现 dnslog 回连平台
